两个整数最小公倍数相关定理的证明

Table of Contents

文章仅供备忘，所以排版格式不是很规范

1. 设两个互质的数分别为 a，b ，证明其最小公倍数为 ab

反证：假设存在 K < ab 为 a, b 的最小公倍数。则 K/a，K/b 为整数，不妨设为 m,n。结合 K < ab 的假设。有：

m = K/a < ab/a = b , n = K/b < ab/b < b

又

a · m = a · K/a = K = K/b · b = n · b

得

am/b = n

由于 a，b 互质，且 m < b. 所以等式左边不为整数。而等式右边是整数。

矛盾，故假设不成立。

2. 设 a, b 为整数。其最大公约数为 s. 则其最小公倍数为 ab/s

设 a = sx, b = sy

反证：假设存在 K < ab/s = sxy 为 a, b 的最小公倍数则 K/a ，K/b 为整数，不妨设为 m,n ，则有

m = K/a < sxy/sx = y n = K/b < sxy/sy = x

且

a · m = a · K/a = K = K/b · b = n · b

» sxm = am = nsy » xm=ny » m = ny/x

因为 x,y 互质，且 n < x, 右边不为整数。而左边为整数。

矛盾，故假设不成立。
